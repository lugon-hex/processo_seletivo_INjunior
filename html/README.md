# processo-seletivo-in-junior-html

> Projeto em HTML feito para o processo seletivo da {IN}Junior, empresa júnior de Computação da UFF, com páginas de apresentação dos integrantes, formulários de contato e conteúdo multimídia.

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)](https://developer.mozilla.org/pt-BR/docs/Web/HTML)

---

## Sobre o projeto

Este projeto é um site estático desenvolvido em HTML para apresentar os integrantes do grupo, reunir informações pessoais e disponibilizar páginas de contato. A proposta segue o contexto do processo seletivo da {IN}Junior, com navegação simples entre as páginas principais, perfis individuais e formulários de feedback.

---

## Estrutura

```
HTML/
├── main_page.html          # Página inicial do grupo
├── pedro_page.html         # Perfil do Pedro Lucas
├── pedro_contato.html      # Página de contato do Pedro Lucas
├── fabio_page.html         # Perfil do Fábio Luiz
├── fabio_contato.html      # Página de contato do Fábio Luiz
└── assets/                 # Imagens e vídeos usados no site
```

| Ferramenta | Uso |

---

## Como executar

### 1. Abra o projeto
Abra a pasta do repositório em um navegador de arquivos ou no VS Code.

### 2. Inicie pela página principal
Abra o arquivo `main_page.html` no navegador.

### 3. Navegue pelas páginas
Use os links da página inicial para acessar os perfis e as páginas de contato.

> Como o projeto é estático, não é necessário instalar dependências nem iniciar servidor para visualizar o conteúdo.
